General Learning Links

  • hackr.io - A categorized list of resources for every language and framework imaginable. Users submit a resource and then it receives up or down votes based on its quality.
  • Stack Overflow - A must-have for all developers. Users submit questions under a variety of categories and receive responses ordered by quality.
  • Udemy (paid) - Buy professional-grade courses on any topic. There are often steep discounts available. If you can’t find it for free, it’s on Udemy for a reasonable price.
  • MDN Web Docs - A comprehensive documentation and tutorial site for web development, by Mozilla. These docs are infinitely more valuable and fleshed out than their W3 Schools counterparts.
